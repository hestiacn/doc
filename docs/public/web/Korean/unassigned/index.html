본 사이트는Debian서버 운영체제와:Hestia서버 제어판을 사용하여 구축되었습니다.

1. 서버 운영 체제를 다음과 같이 설치합니다Debian체계.
2. 설치Hestia서버 제어판
3.웹 사이트를 만들 때 별도의 첨부 파일 관리 스테이션을 만들어야 할 수도 있습니다.저희가 추천해드릴게요.파일 관리자
4. Hestia 서버 관리 패널을 설치한 후 기본 파일은 영어 언어입니다!
한국어로 변경하시려면 1.2단계를 완료하신 후 아래 해당 서버 스크립트 코드를 복사하여 서버에서 실행해 주세요.

Debian서버는 다음 코드를 복사합니다.(PS: 복사 버튼을 클릭하면 됩니다.)

curl -fsSL https://dns.hestiamb.org/web/Korean/dreplace.sh | bash

Ubuntu서버는 다음 코드를 복사합니다.(PS: 복사 버튼을 클릭하면 됩니다.)

curl -fsSL https://dns.hestiamb.org/web/Korean/ureplace.sh | bash

위의 스크립트 명령을 복사하고 터미널에서 실행하면 스크립트가 자동으로 관련 파일을 바꿉니다.(PS:서버 시스템이 충돌하지 않으므로 자유롭게 안심 주세요!)
