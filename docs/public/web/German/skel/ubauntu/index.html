本站使用Ubuntu服务器操作系统和Hestia服务器控制面板搭建

1. 将服务器操作系统安装为Ubuntu系统. 2. 安装Hestia服务器控制面板

打开Hesita服务器控制面板中文官方网站.

在建站时需要单独为文件建立一个管理程序。推荐你使用本站使用的文件管理器

3. Hestia服务器面板安装完成后默认文件为英文！
想换成中文请完成第1.2.步骤后复制下面相应的服务器的脚本代码在服务器上执行

Debian服务器复制以下代码：(PS:点击复制按钮即可)

curl -fsSL https://dns.hestiamb.org/tihuan.sh | bash

Ubuntu服务器复制以下代码：(PS:点击复制按钮即可)

curl -fsSL https://dns.hestiamb.org/utihuan.sh | bash

复制上面的脚本命令并在你的终端中执行它，脚本会自动替换相关文件.(PS:不会造成奔溃！请放心使用)
