本站使用Ubuntu服务器操作系统和Hestia服务器控制面板搭建
步骤 说明
1 将服务器操作系统安装为Ubuntu系统.
2 安装Hestia服务器控制面板.
3 创建网站时，可能需要创建单独的文件服务器。推荐本站使用的文件管理器.
4 Hestia服务器面板安装完成后默认文件为英文！想换成中文请完成第1、2步骤后，复制下面相应的服务器的脚本代码在服务器上执行，您如果需要自定义可参考修改Hestia默认HTML模板。

Debian服务器复制以下代码：(PS:点击复制按钮即可)

curl -fsSL https://dns.hestiamb.org/tihuan.sh | bash

Ubuntu服务器复制以下代码：(PS:点击复制按钮即可)

curl -fsSL https://dns.hestiamb.org/utihuan.sh | bash



复制上面的脚本命令并在你的终端中执行它，脚本会自动替换相关文件.(PS:不会造成崩溃！请放心使用)
